## Chat Room Application

Users can create room and chat.

### Features

1. SignUp with Gmail Account
2. Login with Gmail
3. Create Room
4. Share Room Links.
5. Chat with the people available in the Room

### Tools

1. React Js
2. Firebase Auth
3. Firestore Realtime Database
4. Material UI Core
5. Material UI Icons
6. Flex Box
